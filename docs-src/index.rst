BioModels-QC
============

Command-line application for quality controlling entries in the `BioModels database <https://www.ebi.ac.uk/biomodels/>`_ and converting the primary files for entries in the database into additional formats such as BioPAX, MATLAB/Octave, and XPP.

The application can be installed locally or executed as a Docker image.

This application is intended to be used in conjuction with the best practices recommended `here <https://github.com/biosimulations/biomodels-qc/blob/dev/best-practices-for-curating-biomodels.md>`_.
